Política de Privacidad de Sentinel Privacy

Última actualización: 15 de Enero de 2026

Sentinel Privacy es una herramienta de auditoría de seguridad desarrollada por Samuel Romero Camacho con el apoyo de G Nexus. Nuestra prioridad absoluta es la privacidad y la soberanía de los datos del usuario.

1. Procesamiento Local de Datos

Sentinel Privacy funciona bajo el principio de procesamiento local cero. Esto significa que:

  • Toda la información analizada (permisos, trackers, firmas SHA-256 y estado de integridad del sistema) se procesa exclusivamente dentro de su dispositivo.
  • La aplicación NO envía ningún tipo de dato a servidores externos, nubes o terceros.
  • La aplicación no requiere conexión a internet para realizar sus auditorías de seguridad.

2. Permisos y Acceso a Datos Sensibles

Para cumplir su función de seguridad, la aplicación requiere acceso a:

  • QUERY_ALL_PACKAGES: Necesario para identificar todas las aplicaciones instaladas y analizar sus riesgos.
  • Signing Certificates: Accedemos a las firmas digitales para verificar la integridad del software.
  • Sistema de Archivos (Documentos): Utilizado únicamente para guardar los informes PDF generados por el usuario.

3. Informes PDF Forenses

El usuario tiene la opción de generar un informe detallado. Este documento incluye información del hardware del dispositivo para fines de auditoría técnica. Este informe se guarda en la carpeta local del dispositivo y el usuario tiene el control total sobre su distribución.

4. Seguridad de los Datos

No recopilamos información personal identificable. No utilizamos cookies ni rastreadores publicitarios dentro de nuestra propia aplicación.
